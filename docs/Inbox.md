# Inbox

Quick capture for ideas, issues, and research before organizing them into proper documentation.

## How to Use

1. Add a bullet point below when an idea comes up while working.
2. Review weekly and move items to the correct location:
   - Feature ideas → `plans/`
   - Decisions → `docs/decisions/`
   - Bugs → create a Bug Investigation note
   - Notes → `docs/notes/`
3. Delete items after they have been moved.
